# RubiChess Analysis Suite

This directory contains all analysis scripts, reports, and data files related to RubiChess engine evaluation and optimization.

## Directory Structure

- `analysis_results/` - CSV files with detailed analysis results
- `*.py` - Python analysis scripts for engine comparison and testing
- `*.md` - Analysis reports and documentation
- `*.json` - Detailed analysis data in JSON format
- `*.csv` - Summary analysis results
- `*.pgn` - Chess position files for testing

## Key Analysis Scripts

- `deep_dive_analysis.py` - Comprehensive engine analysis
- `engine_comparison.py` - Compare RubiChess with other engines
- `large_scale_engine_comparison.py` - Large-scale performance testing
- `debug_engine_analysis.py` - Debug and troubleshooting tools
- `phase1_emergency_fixes.py` - Critical bug fixes and optimizations

## Reports

- `phase1_completion_report.md` - Phase 1 analysis completion summary
- `large_scale_weakspot_report.md` - Identified weaknesses and areas for improvement
- `critical_optimization_roadmap.md` - Optimization strategy and roadmap
- `comprehensive_weakspot_analysis.md` - Detailed weakness analysis

## Usage

Run analysis scripts from this directory. Most scripts are designed to work with the RubiChess engine located in `../RubiChess/`.

Example:
```bash
cd analysis
python deep_dive_analysis.py
```
